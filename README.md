# Financeiro TECH NET Android

Projeto Android nativo em Kotlin + Jetpack Compose, preparado para compilar **direto no GitHub Actions**.

## Como usar só pelo GitHub

1. Crie um repositório no GitHub.
2. Extraia este ZIP.
3. Envie todos os arquivos para a raiz do repositório.
4. Vá em **Actions**.
5. Execute o workflow **Build Android APK**.
6. Baixe o APK em **Artifacts**.

## Observação
- Esta base usa dados simulados.
- O próximo passo é conectar o app ao seu sistema web via API PHP.
